--- layout: component title: Billboard description: A Billboard is a hero panel used to feature an important message and call to action at the top of a page, typically the home page of a website. Billboards may use an image or graphic as the background to aid users in understanding the message and user desired action. guidelines: do: - Keep title and description brief - Use a clear call to action to navigate to another page, screen, or UI - Use when it is preferred that the presentation is visual in nature - Add a color overlay when using a background image, for improved readability dont: - Include multiple billboards on a page - Link to direct downloads as the call to action - Use call to action to initiate a command or any process that is not navigation - Include ambiguous, distracting, or unrelated background images contents: - Usage guidelines - Basic Billboard - Fluid Billboard - Dark background - Background images ---
