--- layout: component title: Card description: A card is a flexible and extensible content container. It includes options for headers and footers and a wide variety of content. Like many other components, cards are fluid. Their width can be organized by putting them inside grid columns. If you want your cards to be equal height, use a card deck or ensure that the card content is the same height. The card also acts as a link to a detailed view of the content. In fact, it should always drill down to more detailed information. guidelines: do: - Use to demonstrate a set of KPIs - Use to provide at-a-glance content - Use in the context of an Overview or Dashboard dont: - Use for featured content - Use for extensive content - Use as a button for executing a command - Use as primary navigation contents: - Usage guidelines - Cards with grid - Card deck ---
