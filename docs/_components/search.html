--- layout: component title: Search description: Use this component to show search results as users type (styles only). description: The search component is a specialized type of input field that enables a query against a data set. It can be assigned a global or contextual scope. It includes a search icon in the input field and has an optional search results list to display results below the search box as the user types. The query is initiated by pressing the ENTER key after entering the query string, or by pressing a "search" button. guidelines: do: - Use when there is a large data set that the user should be able to explore in detail dont: - Use when the data set is small enough to browse or enumerate in a list view (such as a table) - Use as a substitute for navigation contents: - Usage guidelines - Basic search box - Inline results - Dropdown results - Dropdown result metadata ---
