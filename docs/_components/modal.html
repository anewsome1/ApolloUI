--- layout: component title: Modal description: A Modal is a secondary window that appears on top of the main UI and must be dismissed or engaged before it allows the user to return to the main UI. Modals are primarily meant as very brief interludes in the user experience to either present featured content or offer up a targeted interaction to get work done within the scope of the main task or activity. guidelines: do: - Use to direct the user to featured content - Use to direct the user's attention to a sub-task without taking them away from the current screen dont: - Use for extended activities or tasks - Use to show alerts, error, success, or warning messages - Use on mobile devices - Use for multi-step processes - Make content scroll within the modal window contents: - Usage guidelines - Basic modal - Small size - Large size - Login modal ---
